## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

This is a fully static client-side app. Deploy to Vercel with zero config:

```bash
npm run build
npx vercel deploy
```

No environment variables or database setup required.
